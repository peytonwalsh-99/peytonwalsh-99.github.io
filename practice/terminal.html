How to use vim

Moving Around the Screen

  • Pressing "h" moves left
  • Pressing "l" moves right
  • "j" moves down
  • "k" moves up
  • "w" jumps forward to the start of a word
  • "e" jumps forward to the end of a word
  • "b" jumps backwards to the start of a word
  • "ge" jumps backward to the end of a word
  • "$" jumps to the end of the line
  • You can press a number before one of these keys to jump spaces. Such as "6k" will jump 6 lines up.

Opening and closeing files

To open a file in vim, you must go through the terminal. When you are in the correct folder, type vim [filename]. You can also create files by simple typeing a name of a file that does not exist yet.

  • Typing :wq allows you to save and close the file.
  • Typing just :q allows you to quit without saving.
    This may give you an error message, if you really want to quit without saving type :q!
  • ":w lets to save without leaving
  • ":wqa" - save and quit all tabs

Editing

  • r-replace single character
  • i-insert before cursor
  • a-insert after cursor
  • o-open new line below current line
  • O-open new line above current line
