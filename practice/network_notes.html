Internet protocol intro

Protocol = The rules that govern how data is transmitted and received

Packet = Small units which data is transmitted in

Header = The block of data that contains the packet's information

Payload = The part of the packet with the actual message

Router = A device that directs data traffic, also is what connects your computer to the internet

Routing Table = A datatable thats used for storing information about routes and networks, a map of nearby routers.

The main problems that can occur are potential delys due to congestion (too much data at once) or lost from congestion or hardware issues.
