Overview of the courses

  • DE Web Design - Focused on basic html, css and javascript code. Offers students a chance to earn a certification in html. Focused on the fundementals of coding and the internet.

  • Programming I - Provides an intro to python, completes and industry certification in python. Intro to full stack web development using the Django web framwork. Earns 6 college credits.

  • Programming II - Focuses on C++ and Discrete Math. Completes a certification in C++. Earns 7 college credits.

  • Programming III - Computer Architecture, using the Altair_8800 and algorithms earns 7 college credits.

  • DE Database Design & Management - Focuses on understanding data collection, data warehousing, analytics and presentation of findings. Students will work to optimize connections between databases and modern websites, as well as interacting with databases through python ORMs (object-Relational Mappings).
